## このプロジェクトは何か

Hans Rosling『FACTFULNESS』のフォーマット × 東洋哲学の Boolean→Float フレームで、
**「世界は思っているのと違う素材でできている」** をデータと物語で描く11章構成の書籍プロジェクト。

**中核コンセプト：**
- 素材は「良い・悪い」ではなく連続値（float）の集合
- 文明とは「元素を並べ直す技術」の集大成
- インタラクティブ図鑑（Web）と本（物語）のメビウスの帯構造

## デプロイ標準パターン（v4で確立）

1. 原稿 md を Python markdown ライブラリで HTML 変換
2. ch1テンプレートのCSSを `--accent` だけ差し替えて流用
3. h2セクション直後に `<figure>` を自動挿入（needle文字列マッチ）
4. JPEG圧縮（1200px幅 quality78、章あたり1MB前後）
5. GitHub Contents API で `{ch}/index.html` + `{ch}/images/` をPUT
6. メイン `index.html` ナビバナーに章リンク追加
7. GitHub Pages伝播1〜2分待ってLIVE確認

このパターンはv4セッションで4章分（序/Ch8/Ch9/終）を一気にデプロイして検証済み。
